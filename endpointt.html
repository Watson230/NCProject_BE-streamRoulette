Stream Roulette Api Documentation

Films

Get

/api/films/:id

finds film with specified Id

/api/film/liked

gives array of the 10 most films with the most likes

/api/film/disliked

gives array of the 10 most films with the most dislikes

/api/film/watched

gives array of the 10 most watched films

PUT

/api/films/:id/likes

updates a films like count

/api/films/:id/dislikes

updates a films dislike count

/api/films/:id/watched

updates a films watched count

Post

/api/film

saves a new film to DB. requieres film object in req body

Search Queires

PUT

/api/search/results/liked/:userId

adds a film to a users liked films

/api/search/results/disliked/:userId

adds a film to a users disliked films

/api/search/results/watched/:userId

adds a film to a users watched films

/api/search/:userId/queires

saves a users search quieres

User

GET

/api/user/:username

gets full viewing and search history of a user

POST

/api/user

Saves new user to database, requires user object in req.body
